My Learning Journey

A deeper look into my development path

How I learn best

Iam a hands on learner. I can not just watch tutorials and remember everything I need to build things myself and break them to understand how they work. Whenever I learn something new, I immediately try to apply it to a real project.

For example, when I first learned about CSS flexbox, I completely rebuilt my navigation bar from scratch using only flexbox. It broke everything initially, but fixing those mistakes taught me more than any tutorial could.
