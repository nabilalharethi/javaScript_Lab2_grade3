# javaScript_Lab2_grade3

# A client-side contact form with comprehensive validation, real-time feedback, and responsive design. Built with vanilla JavaScript (ES6+), CSS3, and HTML5. No external dependencies required.

## Installation & Setup

### Prerequisites

Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
Text editor (VS Code, Sublime Text, Atom, etc.)

# 1. Clone or download the repository
git clone <repository-url>

# 2. Navigate to project directory
cd portfolio-contact-form

# 3. Open in browser (no build required)
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html

# Usage Guide

## Accessing the Contact Form
### Method 1: Direct Access
file:///path/to/project/contact.html
### Method 2: Through Navigation

Open index.html
Click "Contact" in navigation menu
Form loads at /contact.html
